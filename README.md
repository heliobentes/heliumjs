# Helium Monorepo

This repository contains the source code for the Helium framework and an example application.

## Structure

-   **`helium/`**: The core Helium framework package. See [helium/README.md](./helium/README.md) for documentation.
-   **`example-app/`**: A demo application showcasing how to use Helium.

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Build the framework:
    ```bash
    cd helium
    npm run build
    ```

3.  Run the example app:
    ```bash
    cd example-app
    npm run dev
    ```
